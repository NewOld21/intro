# 🏆 My Sports Archive
## 나의 스포츠 아카이브

> **스포츠를 넘어 삶의 궤적까지, 몰입의 추억으로 채우는 나만의 아카이브**

---

## 📌 Project Overview

**My Sports Archive**는 단순한 스포츠 정보 제공 서비스를 넘어,  
내가 좋아하는 스포츠 팀과 선수에 대한 **개인적인 경험과 추억을 기록하고 공유하는 스포츠 아카이브 웹 서비스**입니다.

스포츠를 처음 좋아하게 된 계기부터 기억에 남는 순간, 현재 응원하는 팀과 선수까지  
한 사람의 팬으로서 변화해온 과정을 하나의 이야기로 기록합니다.

---

| **서비스 분위기 (Mood & Tone)** | **Authentic** : 개인의 경험과 이야기가 담긴 진정성 있는 분위기<br><br>**Dynamic** : 스포츠의 역동성과 열정을 표현<br><br>**Casual** : 누구나 편하게 즐길 수 있는 친근한 분위기 |
| **디자인 방향** | 스포츠 경기장의 에너지와 개인 아카이브 감성을 결합한 디자인 |
| **컬러 시스템** | **Midnight Navy (#0A192F)**<br>깊이 있는 밤 경기장과 스포츠 팬의 몰입감 표현<br><br>**Electric Cyan (#00E5FF)**<br>역동성과 새로운 기록을 표현하는 포인트 컬러 |
| **폰트** | Pretendard |
| **핵심 기능** | - 스포츠 종목별 아카이브 조회<br>- 좋아하는 팀 및 선수 소개<br>- 경기 및 주요 순간 기록<br>- 나만의 스포츠 스토리 작성<br>- 반응형 웹 기반 모바일/PC 지원 |

### 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- Pretendard
- Font Awesome 6.5.1
- Unsplash 이미지
- Local Storage

프레임워크와 빌드 도구 없이 브라우저에서 바로 실행할 수 있도록 구성했습니다.

### 프로젝트 구조

```text
iam/
├─ html/
│  ├─ index.html       # 인트로 및 스포츠 정체성
│  ├─ archive.html     # 종목별 스포츠 아카이브
│  ├─ values.html      # 팬으로서 지향하는 핵심 가치
│  ├─ current.html     # 현재 스포츠 관심사
│  └─ contact.html     # 연락처 및 팬 방명록
├─ css/
│  └─ style.css        # 공통 스타일 및 반응형 디자인
├─ js/
│  └─ main.js          # 메뉴, 필터, 애니메이션, 방명록 기능
└─ Readme.md
```

### 페이지 구성

| 페이지 | 파일 | 주요 내용 |
| --- | --- | --- |
| Intro | `html/index.html` | 서비스 소개, 스포츠의 의미, 열정의 원동력 |
| Archive | `html/archive.html` | Manchester City, KIA Tigers, LA Dodgers, Gen.G, Mercedes-AMG, UFC 기록 |
| Values | `html/values.html` | 서사적 팬심, 능동적 기록자, 변화하는 팬심, 라이프스타일 스포츠 |
| Current | `html/current.html` | SSAFY 교육 중의 스포츠 관심사와 리오넬 메시 헌정 문구 |
| Connect | `html/contact.html` | 사용자 소개와 실시간 한 줄 토크 보드 |

### 주요 기능

- 종목 카테고리별 아카이브 필터링
- 데스크톱 및 모바일 반응형 내비게이션
- 스크롤 위치에 따른 콘텐츠 등장 효과
- 종목에 맞는 배경 이미지와 Font Awesome 아이콘 적용
- 팬 방명록 메시지 등록 및 삭제
- Local Storage를 이용한 방명록 데이터 보존
- HTML 이스케이프 처리를 통한 사용자 입력 보호
- JavaScript가 실행되지 않아도 본문이 표시되는 기본 접근성 처리

### 실행 방법

별도의 설치나 빌드 과정은 필요하지 않습니다.

1. 프로젝트 폴더를 내려받습니다.
2. `html/index.html`을 브라우저에서 엽니다.
3. 로컬 서버 확장 기능을 사용한다면 `html/index.html`을 시작 페이지로 실행합니다.

VS Code에서는 Live Server 등의 정적 서버 확장 기능을 사용할 수 있습니다.

### 데이터 저장 안내

팬 방명록은 별도의 서버나 데이터베이스를 사용하지 않습니다. 작성한 메시지는 브라우저의 Local Storage에 `sports_messages` 키로 저장되므로, 다른 브라우저나 기기에는 자동으로 공유되지 않습니다.
